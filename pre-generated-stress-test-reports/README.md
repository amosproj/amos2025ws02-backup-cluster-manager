# General Stress Test Informations

This folder contains pre-generated stress test reports for various backup volume operations. Each report provides detailed insights into the performance and behavior of the system under different stress conditions.

The report can be read by opening the `results.html` file in ./stress-test-1 and ./stress-test-2 directories inside a web browser.

The first report was based on a Windows 11 System with the following specifications:
- CPU: Intel(R) Core(TM) i7-8700K CPU @ 3.70GHz   3.70 GHz
- GPU: NVIDIA GeForce RTX 3070
- RAM: 32 GB
- Storage: 1 TB NVMe SSD M.2
- OS: Windows 11 Pro
